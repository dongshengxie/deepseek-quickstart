``` ## 游戏功能说明 1. **基本规则**： - 15×15 标准五子棋棋盘 - 黑方先行，轮流落子 - 先形成五子连珠的一方获胜 2. **游戏功能**： - 点击棋盘格子放置棋子 - 自动判断胜负并高亮显示获胜的五子 - 重新开始按钮 - 悔棋功能 3. **界面特点**： - 木质风格的棋盘设计 - 黑白棋子有立体效果 - 清晰的回合提示 - 标准五子棋星位标记 您可以将上述代码复制到一个HTML文件中，然后在浏览器中打开即可开始游戏。游戏完全自包含，不需要任何外部依赖。
